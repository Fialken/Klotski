Foi implementado o jogo Klotski em pygame e usado algoritmos de pesquisa para ajudar/resolver os problemas
Foi implementado no software: BFS, Greedy e A* sendo que apenas usa A* na resolução dos problemas


Contém uma pasta "programas" e um pdf que foi utilizado na apresentação e ao qual pode aceder para entender a linha de reaciocinio por detrás do desenvolvimento dos nossos programas.
Na pasta "Programas" estão os dois ficheiros e uma pasta "Assests" que contém PNGs utilizados por Klotski.py 
Para correr o programa software.py deve usar python 3.10 ou alguma versão mais recente, este programa serve para testar os 3 métodos de pesquisa que implementamos.

O ficheiro "Klotski" deve ser aberto no mesmo diretório da pasta "Assets", pois faz uso da mesma. Este programa contém o programa do ficheiro "software.py" com algumas alterações, bem como todo um trabalho desenvolvido com o pygame para o utilizador poder interagir com o jogo.
Visto que o ficheiro "Klotski" contem toda a programação do ficheiro "software.py", decidimos comentar apenas este.
